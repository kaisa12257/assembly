1. String Primitive Instructions (문자열 기본 명령들)
1. 공통 특징

문자열 명령들은 ESI(소스), EDI(목적지) 레지스터를 자동 사용.

반복 접두사(REP, REPE, REPNE)를 붙이면 ECX만큼 자동 반복.

인덱스 증가/감소는 **Direction Flag(DF)**로 결정.

CLD: 앞으로(+), STD: 뒤로(-)

2. 주요 문자열 명령

MOVSB/W/D : [ESI] → [EDI] 복사, 자동 증가/감소

CMPSB/W/D : 두 메모리(ESI, EDI) 비교

SCASB/W/D : AL/AX/EAX vs [EDI] 비교

STOSB/W/D : AL/AX/EAX → [EDI] 저장

LODSB/W/D : [ESI] → AL/AX/EAX 로드

2. Irvine32 문자열 함수들

여러 문자열 처리 기능을 라이브러리 형태로 제공:

1. Str_length

null 종료 문자열의 길이를 EAX로 반환

2. Str_copy

문자열 복사

3. Str_trim

문자열 끝의 특정 문자 제거
예: "Hello///" → "Hello"

4. Str_ucase

문자열을 모두 대문자로 변환

5. Str_compare

두 문자열 비교

ZERO flag: 같음

CARRY flag: 크기 비교 판단

3. 예제 프로그램(StringDemo.asm)

다음 기능들을 시연:

trim_string: 문자열 끝의 / 제거

upper_case: 문자열을 대문자로 변환

compare_strings: 문자열 비교 결과 출력

print_length: 문자열 길이 출력

4. 2차원 배열(행렬) 처리
1. 주소 계산

2D 배열은 row-major 구조

기본 형식:

[base + index*scale + displacement]

2. 예: 행(row) 합 계산

calc_row_sum:

EBX = 시작 주소

EAX = row index

ECX = row 크기

해당 행의 모든 요소를 더해 EAX에 반환

5. 정수 배열의 탐색 및 정렬에서는
1. Bubble Sort (버블 정렬)

BubbleSort:

이웃한 두 값을 비교하여 큰 값을 뒤로 이동

한 번 반복하면 가장 큰 값이 끝에 위치

n-1번 반복하면 전체가 정렬됨

2. Binary Search (이진 검색)

BinarySearch 기능:

배열이 정렬되어 있다고 가정

first, last, mid를 이용해 검색

중앙값과 비교하여

같으면 mid 반환

찾는 값이 크면 왼쪽 부분 제거

작으면 오른쪽 부분 제거

못 찾으면 -1 반환


따라서
정수 배열을 랜덤 값으로 채움

배열 출력

버블 정렬 수행

다시 배열 출력

사용자에게 정수 입력받음

이진 탐색 수행

찾은 위치를 출력하거나 실패 메시지 출력
